# 📱 مستندات جامع اپلیکیشن GymAI Pro

## 🎯 معرفی کلی

**GymAI Pro** یک پلتفرم جامع و هوشمند برای مدیریت تناسب اندام است که با استفاده از فناوری‌های پیشرفته، تجربه‌ای کامل از تمرین، تغذیه، مربیگری و تعامل اجتماعی را برای کاربران فراهم می‌کند. این اپلیکیشن با معماری مدرن Flutter و بک‌اند Supabase ساخته شده و از یکپارچگی با WordPress برای مدیریت محتوا استفاده می‌کند.

---

## 🏗️ معماری و تکنولوژی‌ها

### **Frontend (اپلیکیشن موبایل)**
- **فریمورک**: Flutter (Dart SDK >=3.8.0)
- **State Management**: Provider Pattern
- **UI Framework**: Material Design با تم سفارشی
- **Responsive Design**: Flutter ScreenUtil + Responsive Framework
- **Localization**: پشتیبانی کامل از فارسی (RTL) و انگلیسی
- **Animation**: انیمیشن‌های سبک و سریع (light animations)

### **Backend & Database**
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth با OTP
- **Real-time**: Supabase Realtime برای چت و نوتیفیکیشن
- **Storage**: Supabase Storage برای فایل‌ها و تصاویر
- **Edge Functions**: Supabase Functions برای پردازش‌های سمت سرور

### **Content Management**
- **CMS**: WordPress REST API
- **Integration**: تمرینات و غذاها از WordPress دریافت می‌شوند
- **Custom Endpoints**: API های سفارشی برای احراز هویت و پرداخت

### **Payment Gateways**
- **درگاه‌های پرداخت**: زیبال (Zibal) و زرین‌پال (ZarinPal)
- **سیستم کیف پول**: کیف پول داخلی با قابلیت شارژ و پرداخت
- **Bridge**: PHP Bridge برای ارتباط بین اپلیکیشن و WordPress

### **Notifications**
- **Firebase Cloud Messaging**: برای نوتیفیکیشن‌های Push
- **Local Notifications**: برای یادآوری‌های محلی
- **Timezone**: پشتیبانی از تایم‌زون تهران (Asia/Tehran)

## 📂 ساختار اصلی اپلیکیشن

### **1. Navigation (نوار ناوبری اصلی)**

اپلیکیشن دارای 5 بخش اصلی در نوار پایین است:

1. **جیم‌آی (AI Hub)** - مرکز هوش مصنوعی
2. **آکادمی** - مقالات و محتوای آموزشی
3. **داشبورد** - صفحه اصلی کاربر
4. **مربیان/رتبه‌بندی** - لیست و رتبه‌بندی مربیان
5. **پروفایل** - تنظیمات و اطلاعات کاربر

### **2. Authentication (احراز هویت)**

#### ویژگی‌ها:
- ✅ ثبت‌نام با شماره تلفن و نام کاربری
- ✅ ورود با شماره تلفن
- ✅ تایید OTP از طریق SMS
- ✅ ایجاد خودکار پروفایل پس از ثبت‌نام
- ✅ مدیریت Session و احراز هویت پایدار
- ✅ پشتیبانی از حالت آفلاین

#### سرویس‌ها:
- `AuthStateService`: مدیریت وضعیت احراز هویت
- `SupabaseService`: عملیات اصلی احراز هویت
- `OTPService`: مدیریت کدهای تایید

## 💰 مدل درآمدی (Monetization Strategy)

### **1. اشتراک‌ها (Subscriptions)**

**اشتراک ماهانه (31 روزه):**
- دسترسی به تمام ویژگی‌های اپلیکیشن
- برنامه‌های AI نامحدود
- دسترسی به آکادمی
- پشتیبانی اولویت‌دار

**اشتراک مربی:**
- دسترسی به داشبورد مربی
- امکان دریافت شاگرد
- ابزارهای پیشرفته مربیگری
- آمار و گزارش‌گیری

### **2. پرداخت برای خدمات (Pay-per-Service)**

**برنامه‌های AI:**
- هر برنامه تمرینی AI: مبلغ مشخص
- هر برنامه رژیمی AI: مبلغ مشخص
- پرداخت یکباره یا از کیف پول

**خدمات مربی:**
- برنامه تمرینی شخصی: مبلغ مشخص
- برنامه رژیمی شخصی: مبلغ مشخص
- مشاوره آنلاین: مبلغ مشخص
- اشتراک ماهانه مربی: مبلغ مشخص

### **3. سیستم کیف پول (Wallet System)**

**مزایا:**
- پرداخت سریع‌تر
- امکان شارژ با مبالغ مختلف
- تخفیف برای شارژهای بیشتر
- مدیریت بهتر هزینه‌ها

### **4. کمیسیون پلتفرم (Platform Commission)**

**برای مربیان:**
- درصدی از درآمد مربیان از طریق پلتفرم
- مدل درآمدی پایدار

### **5. کدهای تخفیف و تبلیغات**

**استراتژی:**
- کدهای تخفیف برای جذب کاربران جدید
- تبلیغات درون‌برنامه‌ای (در آینده)
- برنامه‌های وفاداری

## 🔄 جریان کارهای اصلی

### **1. ثبت‌نام و ورود:**
1. کاربر شماره تلفن و نام کاربری وارد می‌کند
2. کد OTP ارسال می‌شود
3. کاربر کد را تایید می‌کند
4. پروفایل به صورت خودکار ایجاد می‌شود
5. کاربر به داشبورد هدایت می‌شود

### **2. ساخت برنامه تمرینی:**
1. کاربر وارد Workout Plan Builder می‌شود
2. تمرینات را از لیست انتخاب می‌کند
3. ست‌ها، تکرارها و وزن را تنظیم می‌کند
4. روزهای تمرین را تعیین می‌کند
5. برنامه ذخیره می‌شود

### **3. ثبت تمرین:**
1. کاربر وارد Workout Log می‌شود
2. تاریخ تمرین را انتخاب می‌کند
3. تمرینات انجام شده را ثبت می‌کند
4. ست‌ها، تکرارها و وزن را وارد می‌کند
5. تمرین ذخیره می‌شود

### **4. خرید برنامه AI:**
1. کاربر وارد AI Hub می‌شود
2. برنامه AI را انتخاب می‌کند
3. به صفحه پرداخت هدایت می‌شود
4. روش پرداخت را انتخاب می‌کند (کیف پول یا مستقیم)
5. پرداخت انجام می‌شود
6. برنامه در دسترس قرار می‌گیرد

### **5. ارتباط با مربی:**
1. کاربر مربی را از لیست رتبه‌بندی شده انتخاب می‌کند
2. درخواست همکاری ارسال می‌شود
3. مربی درخواست را تایید/رد می‌کند
4. رابطه فعال می‌شود
5. کاربر می‌تواند چت کند و برنامه دریافت کند
