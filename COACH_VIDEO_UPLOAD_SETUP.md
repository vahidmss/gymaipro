# راهنمای نصب سیستم آپلود ویدیو مربی

## 📋 خلاصه

این سیستم به مربیان و ادمین‌ها امکان آپلود ویدیو تمرین اختصاصی رو می‌ده که مستقیماً روی هاست دانلود (`dl.gymaipro.ir`) ذخیره می‌شه.

## 🔧 مراحل نصب

### 1. نصب فایل PHP روی هاست دانلود

فایل `lib/services/coach_video_upload_standalone.php` رو با نام `upload-video.php` در مسیر زیر قرار بده:

```
/domains/dl.gymaipro.ir/public_html/upload-video.php
```

**نکته مهم:** این فایل باید مستقیماً روی هاست دانلود باشه، نه روی هاست WordPress!

### 2. ساخت پوشه coaches_video

پوشه `coaches_video` باید در مسیر زیر وجود داشته باشه:

```
/domains/dl.gymaipro.ir/public_html/coaches_video/
```

اگر وجود نداره، خودش ساخته می‌شه، ولی بهتره از قبل بسازیش با مجوز `755`:

```bash
mkdir -p /domains/dl.gymaipro.ir/public_html/coaches_video
chmod 755 /domains/dl.gymaipro.ir/public_html/coaches_video
```

### 3. تنظیمات CORS (اختیاری)

اگر از دامنه‌های مختلف درخواست می‌فرستی، ممکنه نیاز به تنظیم CORS باشه. فایل PHP قبلاً CORS رو فعال کرده، ولی اگه مشکل داشتی، می‌تونی در `.htaccess` یا تنظیمات سرور اضافه کنی.

### 4. تست در اپلیکیشن

1. با اکانت **admin** یا **trainer** وارد شو
2. به **میز کار مربی** برو
3. تب **خدمات** رو باز کن
4. روی کارت **"تست آپلود ویدیو"** کلیک کن
5. یک ویدیو انتخاب کن و آپلود کن

## 📁 ساختار فایل‌ها

```
dl.gymaipro.ir/
├── public_html/
│   ├── upload-video.php          ← فایل آپلود (اینجا قرار بگیره)
│   └── coaches_video/            ← پوشه ویدیوها
│       ├── {trainer_id_1}/       ← پوشه هر مربی
│       │   ├── 1234567890_abc123.mp4
│       │   └── ...
│       ├── {trainer_id_2}/
│       └── ...
```

## 🔒 امنیت

- ✅ احراز هویت با JWT token از Supabase
- ✅ بررسی role (فقط admin و trainer)
- ✅ بررسی نوع فایل (فقط MP4, MOV, AVI, WEBM)
- ✅ بررسی حجم فایل (حداکثر 100MB)
- ✅ هر مربی فقط به پوشه خودش دسترسی داره

## 📝 تنظیمات قابل تغییر

در فایل `upload-video.php` می‌تونی این موارد رو تغییر بدی:

- **حداکثر حجم فایل:** خط 147 (پیش‌فرض: 100MB)
- **فرمت‌های مجاز:** خط 139 (پیش‌فرض: mp4, mov, avi, webm)
- **مسیر پایه:** خط 150 (پیش‌فرض: `__DIR__ . '/coaches_video'`)

## 🐛 عیب‌یابی

### خطای 401 (Unauthorized)
- بررسی کن که JWT token معتبر باشه
- بررسی کن که کاربر لاگین باشه

### خطای 403 (Forbidden)
- بررسی کن که role کاربر `admin` یا `trainer` باشه

### خطای 500 (Directory Error)
- بررسی کن که پوشه `coaches_video` وجود داشته باشه
- بررسی کن که مجوزهای پوشه درست باشه (755)

### خطای آپلود فایل
- بررسی کن که حجم فایل کمتر از 100MB باشه
- بررسی کن که فرمت فایل مجاز باشه
