## 🐛 **مشکلات احتمالی و راه‌حل**

### **مشکل: "relation does not exist"**
**راه‌حل:** اجرای migration های دیتابیس
```sql
-- اجرای فایل trainer_system_final_setup.sql
```

### **مشکل: "permission denied"**
**راه‌حل:** بررسی RLS policies
```sql
-- بررسی policies
SELECT * FROM pg_policies WHERE tablename = 'trainer_clients';
```

### **مشکل: "user not found"**
**راه‌حل:** اطمینان از وجود کاربر در جدول profiles
```sql
-- بررسی کاربر
SELECT * FROM profiles WHERE role = 'trainer';
```

### **مشکل: نقش کاربر نمایش داده نمی‌شود**
**راه‌حل:** بررسی فیلد role در جدول profiles
```sql
-- به‌روزرسانی نقش کاربر
UPDATE profiles SET role = 'trainer' WHERE id = 'user_id';
```
